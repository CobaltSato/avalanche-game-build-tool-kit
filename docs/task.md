各フェーズの完了ごとに、実装内容を確認し、検収をお願いいたします。

- [x] プレイヤーを表す `Player` コンポーネントの作成
- [x] 作成したコンポーネントを `app/page.tsx` に配置して表示

## フェーズ 3: コントラクトとの読み取り連携 (未着手)

- [ ] トランザクション成功後、コントラクトから最新の座標を再取得して画面を同期
